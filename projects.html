02 Full-Stack Frontend App

TaskFlow — Kanban Task Manager

A modern, production-style Kanban task management app built with Angular 17. Features drag-and-drop columns, dark/light mode, a live dashboard with analytics, full task CRUD, overdue detection, and localStorage persistence — no backend required.

Technologies Used:
Angular 17 TypeScript Angular CDK RxJS CSS Variables
My Role:

Sole developer. Designed and built the full application architecture including auth flow, state management with BehaviorSubject, drag-and-drop board, and responsive UI with dark/light theming.

Key Contributions:
  • Built Kanban board with CDK drag-and-drop and auto status updates
  • Implemented authentication with route guards and localStorage
  • Created dashboard with live stats, priority breakdown, and canvas donut chart
  • Designed full dark/light theme system using CSS custom properties
  • Deployed to Netlify with CI/CD via GitHub integration

03 UI/UX Design · Figma

MD Movies – Streaming Platform UI/UX Concept & Interactive Prototype

MD Movies is a high-fidelity UI/UX concept for a movie streaming platform, designed entirely in Figma. The project focuses on user journey planning, structured wireframing, visual hierarchy, and interface consistency. It includes a fully interactive prototype that simulates real user navigation and screen transitions. This project is a design mockup and prototype, not a developed or responsive website.

Tools & Methods:
Figma Wireframing Prototyping User Flows Design Systems
My Role:

UI/UX Designer. Planned user flows, designed core interface screens such as homepage, movie detail pages, and navigation layouts, developed a cohesive typography and color system, and built interactive transitions to demonstrate user interaction and usability.

Key Contributions:
  • Designed high-fidelity streaming platform interface mockups
  • Established brand identity including color palette and typography hierarchy
  • Created clickable interactive prototype for presentation and testing
  • Applied modern UI principles focused on clarity, spacing, and usability
